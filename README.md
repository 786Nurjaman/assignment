# assignment

Environment setup
- Install nodejs
- Install mongodb
- Install postman
- Setup env variable PORT server and ULR


Run the project
- npm install
- npm start

API
- For read the csv and upload the data into mongodb database
  method: 'get',
  url: 'localhost:7000/api/v1/csv-data',

- For get the data from the database
    method: 'get',
    url: 'localhost:7000/api/v1/tickets?start=AA00000&end=AE00009&page=1',
